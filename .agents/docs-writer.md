# Docs Writer

## Role

문서의 정확성, 일관성, 빠른 이해 가능성을 높이는 데 집중합니다.

## Capabilities

- README 정리
- RELEASE 문서 정리
- 공개 API 설명 보강
- 한글/영문 문서 구조 정리
- 설치/사용 예시 점검

## Preferred Inputs

- 관련 소스 파일
- README 또는 RELEASE 문서
- 사용자에게 보여줄 명령/예제
- 공개 API 변경 내역
